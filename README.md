Submit, track, and manage your leave requests with ease. Employees can quickly enter leave type, dates, and reason. Managers receive instant approval notifications, and decisions are updated in real time. Stay informed with automatic status updates and notifications through Teams or email.

# 📱 Leave Request Management System
A custom Microsoft Power Apps Canvas application designed to streamline employee leave requests and approval workflows, replacing manual spreadsheet tracking with an automated, data‑driven process.

## ⚙️ App Features & Tech Stack
- **Data Architecture:** Backed by **Dataverse** for structured relational data storage and validation.
- **Key Capabilities:** Interactive form submission, approval tracking, conditional formatting, and Power Automate integration for automated notifications.
- **Key Logic:** Developed using Power Fx formulas including `Patch()`, `Filter()`, and collection‑based data handling for dynamic record updates.
- **UI Design:** Minimalist corporate blue‑teal palette with responsive layout and icon‑based navigation.

## 📂 How to Deploy This App
1. Download the `.msapp` file included in this repository.  
2. Open the [Power Apps Maker Portal](https://make.powerapps.com).  
3. Select **Apps > Import canvas app** and upload the package file.
